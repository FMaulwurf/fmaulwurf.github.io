Arbeit

Angestellt als iOS-Developer für die Mevango GmbH seit April 2015.

Ich bin verantwortlich für die Entwicklung einer Live Streaming Applikation, die als Werkzeug für das Flimme Systems dient. Die Applikation ist ein Hybrid aus nativem Swift Code und React Native. Die Kommunikation findet über eine REST Schnittstelle mit dem Flimme Backend statt. Der Aufbau basiert auf dem Redux Paradigma. Das User Interface ist mit React Native Components umgesetzt, deren Zustände von einer komponentenbasierten State Machine gesteuert werden.
